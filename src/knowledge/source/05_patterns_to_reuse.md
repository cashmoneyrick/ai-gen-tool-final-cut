# Patterns to Reuse

Pattern:
Hard-source design reference plus secondary pose reference
Best use case:
Translating an existing nail set into a new lifestyle shot or controlled scene
Example structure:
"Reference 1 controls nail design, shape, color, art placement, and finish. Reference 2 controls pose, framing, lighting, crop, and background only."
Notes:
This is the default multi-reference pattern and should be stated explicitly.

Pattern:
Change one variable, lock everything else
Best use case:
Collection work, especially cateye monthly color swaps
Example structure:
"Use the approved previous output as the exact camera angle, reflection behavior, and magnetic geometry reference. Change only the colorway."
Notes:
Excellent for month-to-month consistency and drift reduction.

Pattern:
World-fragment packaging construction
Best use case:
Premium embroidered pouch scenes
Example structure:
"Show a medium or hard-cropped fragment from a larger world, with elements continuing off the edges and no strict badge or framed panel."
Notes:
This prevents emblem drift and makes the design feel more authored.

Pattern:
Embroidery-first material control
Best use case:
Any pouch scene that risks looking printed or illustrated
Example structure:
"Visible stitch direction, chunkier satin-stitch fills, raised thread relief, thread-built foliage, stitched ripple bands, slightly simplified forms so the embroidery texture dominates."
Notes:
Use when the scene looks too smooth or painted.

Pattern:
Subtle-branding-by-geometry
Best use case:
Hidden YPS details
Example structure:
"Build YPS from the same ripple, vine, reed, ironwork, or star logic as the surrounding scene so it is discoverable but not written on top."
Notes:
Best when the lettering is nearly swallowed by the native scene system.

Pattern:
Anti-drift keep-and-avoid block
Best use case:
Reference-driven nail generations and cateye control
Example structure:
"Must keep: exact motifs, shape, palette, 3D gel detail, handmade feel. Avoid: redesign, flattening, generic flowers, angle drift, harsh glare."
Notes:
Useful whenever the model tends to simplify or overinterpret.

Pattern:
Filled-field cateye wording
Best use case:
Preventing donut, halo, or spotlight failures
Example structure:
"One continuous filled vertical magnetic field occupying roughly the central 55 to 65 percent of the nail width, with no outline, no hollow center, no ring, and feathered transition into darker edges."
Notes:
This is the key cateye geometry pattern.

Pattern:
Approved-output promotion
Best use case:
Iterative workflows after partial success
Example structure:
"This approved output is now the current source of truth. Keep composition, spacing, lighting, and all other nails locked. Change only X."
Notes:
Very effective for local corrections and consistency.

Pattern:
Visible-only reduction
Best use case:
Incomplete source imagery
Example structure:
"Use only the clearly visible nails as source truth. Do not guess the unseen nails."
Notes:
Reduces invented details and false continuity.

Pattern:
Realism-through-restraint
Best use case:
Handmade nails and product images that risk feeling too polished
Example structure:
"Keep believable gel thickness, slight asymmetry, realistic sidewalls, controlled but not sterile gloss, and minor handmade irregularity."
Notes:
Useful across flatlays, hand shots, and macro single-nail shots.
