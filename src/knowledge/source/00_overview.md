# Image Workflow Knowledge Pack

This pack stores durable operating knowledge for a press-on nail image workflow system with two main image families:

1. Premium embroidered suede pouch packaging mockups
2. Photoreal cateye nail product imagery and nail-reference conversion workflows

It is meant to improve:

* prompt construction
* generation consistency
* drift reduction
* realism
* reference handling
* continuity across sessions, projects, and future tools

Organization:

* `01_hard_rules.md` = rules that should rarely be broken
* `02_preferences.md` = taste defaults and style biases
* `03_workflows.md` = repeatable procedures
* `04_lessons_learned.md` = trial-and-error findings with reuse value
* `05_patterns_to_reuse.md` = reusable prompt and control structures
* `06_examples.md` = short examples of especially valuable logic
* `seed_memory.json` = machine-friendly starter memory for another app

This pack excludes raw chat history and keeps only durable, reusable knowledge.
