# Examples

Example name:
Design ref mandatory, pose ref secondary
Situation:
Turning an existing handmade set into a real-life on-hand image
Decision or prompt tactic:
Make the nail reference the hard source of truth and limit the pose reference to pose, crop, lighting, and background only.
Why it was useful:
It sharply reduced redesign drift.

Example name:
Flatlay conversion with one source image
Situation:
A worn-hand photo needed to become a product flatlay
Decision or prompt tactic:
Use the hand photo as the only source, preserve exact visible nails, and convert to a head-on white row flatlay.
Why it was useful:
It simplified the task and avoided unnecessary contamination from extra references.

Example name:
Edit the approved flatlay, not the original hand shot
Situation:
A flatlay was mostly correct except one charm placement
Decision or prompt tactic:
Promote the approved flatlay to source-of-truth status and edit only the wrong nail.
Why it was useful:
It preserved the already-correct composition and reduced regression.

Example name:
Crop harder to avoid emblem drift
Situation:
Packaging scenes started reading like centered patches
Decision or prompt tactic:
Partially cut off the main subject, let foliage or scene elements continue off the edges, and frame the design like a fragment from a larger artwork.
Why it was useful:
It made the pouch art feel more premium and less generic.

Example name:
YPS hidden as ripple logic
Situation:
Branding needed to be subtle inside a pond scene
Decision or prompt tactic:
Build YPS from the same broken curved ripple segments, same thread color, and same flow as the surrounding water.
Why it was useful:
The branding stayed discoverable without looking stamped on.

Example name:
Filled magnetic field, not a donut
Situation:
A cateye prompt produced a hollow halo shape
Decision or prompt tactic:
Describe the cateye as a continuous filled vertical field with no hollow center, no perimeter line, and gradual feathering into darker edges.
Why it was useful:
It corrected the geometry of the magnetic effect.

Example name:
Use one approved cateye month as the collection anchor
Situation:
Later months were drifting in angle and reflection
Decision or prompt tactic:
Reuse the approved month as the exact camera and highlight reference, changing only the colorway.
Why it was useful:
It stabilized the whole collection.

Example name:
More detail means more depth
Situation:
A richer packaging prompt was needed without becoming cluttered
Decision or prompt tactic:
Increase foreground, midground, background separation and overlap instead of simply adding more objects.
Why it was useful:
It improved richness without killing readability.
